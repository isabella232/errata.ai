  • Native Performance

    Vale compiles into a fast, dependency-free binary available for macOS, Windows, or Linux.

  • Private & Secure

    Vale runs entirely offline—ensuring that your content is never sent to a remote server for processing.

Use Vale where you want

More than just a CLI tool

Vale integrates with third-party apps and services to allow you to enforce you style guide wherever you want.
  • Web-based dashboard

    Create rules, install styles, and manage configurations with Vale Server.

  • Text editor plugins

    Get on-the-fly feedback in your favorite text editor, including VS Code, Sublime Text, Atom, Vim, and Emacs.

  • Browser extensions

    Check your content on the web without having to trust a remote server with Vale's Chrome extension.
